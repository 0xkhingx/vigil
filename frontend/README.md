# Vigil

AI-custodied trader bonds. Stake. Watch. Slash.

Built for the Agora Agents Hackathon — Canteen x Circle x Arc.

## Stack
- Frontend: React + TypeScript
- Contracts: Solidity on Arc
- Backend: Node.js (oracle + agent loop)
- Settlement: USDC on Arc
- AI: Groq API

## Structure
- /frontend — React/TS UI
- /contracts — Solidity slash contracts
- /backend — Oracle + AI agent loop
- /docs — Architecture and notes
